---

## What Finally Made Sense

- `map()` transforms data, `filter()` selects data, and `reduce()` combines data into a single value.
- Closures allow functions to remember variables from their outer scope.
- Immutable updates create new arrays or objects instead of modifying existing ones.
- `Promise.all()` runs multiple asynchronous tasks in parallel, making them faster than running them one by one.
